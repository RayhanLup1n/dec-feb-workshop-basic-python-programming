### Prerequisite

Pastikan Anda sudah memiliki:

- **Python 3.8+** terinstall di komputer ([Download di sini](https://www.python.org/downloads/))
- **Text Editor atau IDE** (Visual Studio Code, PyCharm, Sublime Text, atau apapun yang Anda nyaman)
- Sedikit **motivasi** dan **kesabaran** ğŸ˜Š

### Step-by-Step

1. **Fork repository ini ke akun GitHub Anda**
   - Klik tombol "Fork" di pojok kanan atas repository ini
   - Ini akan membuat copy pribadi dari repository ke akun Anda
   - Gunakan fork Anda untuk bereksperimen dan membuat perubahan

2. **Clone fork Anda ke komputer lokal**

   ```bash
   git clone https://github.com/USERNAME/dec-feb-workshop-basic-python-programming.git
   cd dec-feb-workshop-basic-python-programming
   ```

   _(Ganti `USERNAME` dengan username GitHub Anda)_

3. **Mulai dari folder `00_setup/`**
   - Ikuti instruksi setup
   - Pastikan environment sudah siap

4. **Untuk setiap modul:**
   - Baca file `README.md` di folder tersebut
   - Pahami konsep yang dijelaskan
   - Jalankan file `.py` untuk melihat hasilnya:
     ```bash
     python nama_file.py
     ```
   - **Modifikasi kode** dan eksperimen sendiri!

5. **Jangan skip file README!**
   - Setiap README berisi konteks dan penjelasan mendalam
   - Itu lebih penting daripada kode itu sendiri
